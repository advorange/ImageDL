# ImageDL
Mass download images from websites.

**Currently supports:**
* 4chan
* anime-pictures
* Artstation
* DeviantArt
* Eshuushuu (slow)
* Imgur
* Instagram (likely to have breaking changes in their API soon)
* Moebooru
  * Danbooru
  * Gelbooru
  * Konachan
  * Safebooru
  * Yandere
* Reddit
* Tumblr
* Vsco

*This program is not in any way affiliated with any of the above services.*
